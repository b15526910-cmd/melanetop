Как опытный Frontend-разработчик, я подготовил для тебя современный, адаптивный Landing Page. Этот сайт выполнен в стиле **SaaS-платформы** с использованием **Tailwind CSS** для стилизации и **Lucide Icons** для иконок.

### Основные фишки:
1.  **Glassmorphism эффект**: Современные полупрозрачные элементы.
2.  **Адаптивность**: Идеально отображается на смартфонах, планшетах и ПК.
3.  **Интерактив**: Рабочее мобильное меню, плавная прокрутка и анимации при наведении.
4.  **Чистый код**: Использование современных практик верстки.



### Как запустить:
1. Скопируй код выше.
2. Создай файл `index.html` у себя на компьютере.
3. Вставь код и сохрани.
4. Открой файл в любом браузере (Chrome, Firefox, Safari).

### Что внутри:
*   **Tailwind CSS**: Вся магия стилей без написания громоздкого CSS.
*   **Lucide Icons**: Легкие и красивые векторные иконки.
*   **JavaScript**: Отвечает за динамическое поведение шапки, мобильное меню и простую обработку клика на кнопку.
*   **Дизайн**: Использована современная "темная тема" с акцентами на глубокий синий и фиолетовый цвета.
